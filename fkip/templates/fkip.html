Program studi FKIP merupakan program studi paling banyak diminati pada Seleksi Nasional Masuk Perguruan Tinggi Negeri (SNMPTN), Seleksi Bersama Masuk Perguruan Tinggi Negeri (SBMPTN) menjadi fakultas yang paling banyak di minati oleh calon mahasiswa baru.

FKIP Untirta sudah ada sejak lama, pada awal berdiri FKIP Untirta memiliki dua program studi yakni Program Studi Administrasi Pendidikan (AdPend) dan Program Studi Pendidikan Luar Sekolah (PLS) dengan jenjang pendidikan Sarjana (S.1).

FKIP Untirta terus berkembang sesuai dengan tuntutan masyarakat, sehingga FKIP Untirta kembali membuka program studi yakni Program Pendidikan Guru Taman Kanak-kanak (PGTK) pada 1994, dan beberapa tahun kemudian membuka kembali Program Studi Pendidikan Bahasa Inggris, Pendidikan Biologi dan Pendidikan Matematika. Program studi PGTK telah diubah menjadi Program Pendidikan Guru Pendidikan Anak Usia Dini (PGPAUD) dengan surat keputusan Menteri Pendidikan Nasional Nomor: 105/D/O/2010.
